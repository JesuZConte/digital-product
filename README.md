# digital-product
Demo project for IP Santo Tomas

* Maven version 3.6.0
* Java 8

### To RUN undertow server (runs on port 8080 by default):

a)
BUILD: mvn clean install

b)
mvn spring-boot:run

### For health check:

http://localhost:8080/system/actuator/health

### For info check:

http://localhost:8080/system/actuator/info
